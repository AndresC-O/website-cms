WordPress

¿Qué puedo hacer con WordPress?

En muchas ocasiones se asocia WordPress con una herramienta que solo sirve para hacer blogs. Esto no es correcto: con WordPress podemos crear un blog y mucho más: webs empresariales, tiendas online, periódico digital, central de reservas, etc.


WordPress es un sistema de gestión de contenidos lanzado el 27 de mayo de 2003, enfocado a la creación de cualquier tipo de página web. Originalmente alcanzó una gran popularidad en la creación de blogs, para luego convertirse en una de las principales herramientas para la creación de páginas web comerciales.​


Características de WordPress


  • Simplicidad: La simplicidad hace que sea posible conectarse en línea y publicar rápidamente. Nada debería obstaculizar la creación de tu sitio web y su contenido. WordPress está diseñado para que eso suceda.

  • Flexibilidad: Con WordPress, puedes crear cualquier tipo de web que desees: un blog o una web personal, un fotoblog, una web comercial, un portafolio profesional, una web gubernamental, una revista o web de noticias, una comunidad en línea, incluso una red de webs.

  • Publica con facilidad: Si alguna vez has creado un documento, ya eres un genio para crear contenido con WordPress. Puedes crear publicaciones y páginas, formatearlas fácilmente, insertar medios, y con solo hacer clic en un botón tu contenido estará público y en la web.

  • Herramientas de publicación: WordPress te facilita la administración de tu contenido. Crea borradores, programa su publicación y mira las revisiones de tus publicaciones. Haz que tu contenido sea público o privado, y protege publicaciones y páginas con una contraseña.

  • Gestión de usuarios: No todos requieren el mismo acceso a tu web. Los administradores administran el sitio, los editores trabajan con el contenido, los autores y colaboradores escriben ese contenido, y los suscriptores tienen un perfil que pueden administrar. Esto te permite tener variedad de colaboradores en tu web, y permitir que otros simplemente sean parte de tu comunidad.

  • Gestión de medios: Dicen que una imagen vale más que mil palabras, por lo que es importante que puedas cargar imágenes y medios de manera fácil y rápida en WordPress. Arrastra y suelta tus medios en el cargador para añadirlos a tu web. Añade un texto alternativo, leyendas y títulos, e inserta imágenes y galerías en tu contenido. Incluso hemos añadido algunas herramientas de edición de imágenes con las que vas a disfrutar.

  • Cumplimiento completo de estándares: Cada parte del código generado por WordPress cumple con los estándares establecidos por el W3C. Esto significa que tu web funcionará en el navegador actual, mientras mantenga la compatibilidad con la próxima generación de navegadores.

  • Sistema de temas sencillo: WordPress incluye por defecto tres temas, pero si no son para ti, hay un directorio de temas con miles de temas para que puedas crear una bonita web. ¿Ninguno de esos es de tu gusto? Sube tu propio tema con solo un clic en un botón.

  • Instalación y actualizaciones sencillas: WordPress siempre ha sido fácil de instalar y actualizar. Muchos alojamientos web ofrecen instaladores de WordPress que te permiten instalar WordPress con, bueno, ¡un solo clic! O, si eres feliz usando un programa FTP, puedes crear una base de datos, subir WordPress usando FTP y ejecutar el instalador.

  • Comunidad: Como el CMS de código abierto más popular en la web, WordPress tiene una comunidad vibrante y solidaria. Haz una pregunta en los foros de soporte y obtén ayuda de un voluntario, asiste a una WordCamp o Meetup para aprender más de WordPress, lee artículos en blogs y tutoriales sobre WordPress. La comunidad es el corazón de WordPress, haciéndola lo que es hoy en día.
